# 保留判断リスト

> 実装中にユーザーへの確認が必要な判断を記録します。
> 仮の判断で実装を進め、後でユーザーに確認してまとめて方針を決定します。

**凡例**:
- ⏳ 確認待ち
- ✅ 解決済み（実装に反映済み）
- ❌ 却下（仮判断のまま確定）
