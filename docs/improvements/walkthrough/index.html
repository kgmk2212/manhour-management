このページについて

改善実装が進むたびに、動作確認用のガイドページを追加していきます。 各ガイドは独立しており、確認したい機能だけを試せます。

使い方: 確認したい項目をクリック → 事前準備（テストデータ注入）→ シナリオ通りに操作 → 各ステップのチェック項目を確認。 気になった点があれば decisions-pending.md に記録するか、直接フィードバックいただけます。
