# Product Overview

工数管理システム — ソフトウェア開発プロジェクトの工数（人時・人月）を見積・実績・分析するためのWebアプリケーション。
個人〜小規模チーム向けに、ブラウザだけで完結するローカルファーストな設計。

## Core Capabilities

1. **見積管理** — タスク×工程（UI/PG/PT/IT/ST）のマトリクスで工数を見積もり、版数管理で変更履歴を追跡
2. **実績記録** — 日次の作業実績を記録し、見積との乖離をリアルタイムに可視化
3. **レポート・分析** — 進捗率、精度分析、異常検知、担当者分析などの多角的レポート
4. **スケジュール（ガントチャート）** — Canvas描画によるガントチャートで計画と実績を時系列表示
5. **クイック入力** — 日常の実績入力を簡便にするショートカット画面

## Target Use Cases

- 開発プロジェクトの工数見積と実績トラッキング
- 担当者別・タスク別の工数分析と傾向把握
- プロジェクトの進捗・健全性の可視化

## Value Proposition

- **ゼロインストール**: ブラウザのみで動作、サーバー不要
- **オフライン対応**: localStorageベースでネットワーク不要
- **高いカスタマイズ性**: テーマ、レイアウト、表示モードを豊富に設定可能
- **日本のSI文化に最適化**: 工程（UI/PG/PT/IT/ST）、人月計算、祝日対応
