# レポート分析 リデザイン提案

**ステータス**: 検討中
**作成日**: 2026-03-15
**ブランチ**: experiment/redesign

---

## 概要

レポートタブの分析セクションを再設計し、情報の優先順位を明確化、チャートのスペース効率を改善、分析の多角性を強化する。

## 現状の課題

| # | 課題 | 影響 |
|---|------|------|
| 1 | Phase 1/2/3 の構成に優先順位の根拠がない | 何を最初に見るべきか不明 |
| 2 | メンバー比較棒グラフ・個別ドーナツが大きすぎる | 情報密度が低い、スクロール量大 |
| 3 | 分析の観点が限定的 | バイアス検出、パレート分析、リスク可視化が欠如 |
| 4 | インラインスタイルの色がデザインシステムと不統一 | Bootstrap系カラーと Ink & Amber が混在 |

## 提案する構成

### 情報階層（4層）

```
L1: KPIサマリー ──── 一目で全体把握（4カード）
L2: アラートバー ──── 即アクション対象を集約
L3: 分析ダッシュボード ── 8種のコンパクトチャートカード（2列グリッド）
L4: 詳細データ ──── 担当者別・版数別テーブル（折りたたみ）
```

### L1: KPIサマリー（改善）

- 既存4カード（総見積・総実績・差異・実績率）を維持
- **変更点**: ステータスバッジ追加（「予算内」「良好」等）、スパークライン追加

### L2: アラートバー（新規）

- 超過タスク数・注意タスク数・キャパシティ使用率を1行に集約
- クリックで展開し、具体的なタスク一覧を表示
- 現在の「異常値検出」「要注意タスク」「キャパシティ分析」を統合

### L3: 分析ダッシュボード（大幅改善）

既存の Phase 1/2/3 を廃止し、**目的別のコンパクトチャートカード**に再構成。

| # | チャート | 種類 | 目的 | 新規/既存 |
|---|---------|------|------|----------|
| 1 | 工程別 見積vs実績 | 横棒グラフ | 工程ごとの予実比較 | 既存（コンパクト化） |
| 2 | 見積バイアス分析 | 分岐棒グラフ | 過大/過小見積の傾向検出 | **新規** |
| 3 | 月別推移 | 面+折れ線 | 時系列での予実推移 | 既存（改善） |
| 4 | 工数配分パレート | 棒+累積線 | 上位タスクへの工数集中度 | **新規** |
| 5 | メンバー負荷分布 | 積み上げ横棒 | 負荷バランスと工程配分 | **新規**（既存置換） |
| 6 | リスクヒートマップ | マトリクス | 版数×工程の超過リスク | **新規** |
| 7 | 見積精度トレンド | 折れ線 | 版数ごとの精度改善推移 | **新規** |
| 8 | ステータス分布 | ドーナツ | 全タスクの進行状況 | **新規**（個別ドーナツ統合） |

各カードの構造:
```
┌─ タイトル + サブタイトル ─── バッジ ──┐
│ 凡例（コンパクト）                      │
│ ┌────────────────────────────────┐    │
│ │         チャート描画             │    │
│ │        (高さ 200px)             │    │
│ └────────────────────────────────┘    │
├─────────────────────────────────────┤
│ インサイト: 主要な発見を1行で表示     │
└─────────────────────────────────────┘
```

### L4: 詳細データ（改善）

- 担当者別・版数別テーブルを折りたたみに変更
- ミニ進捗バー付きで視覚的にも把握可能

## 削除・統合する要素

| 現在の要素 | 対応 |
|-----------|------|
| Phase 1/2/3 の折りたたみ構造 | 廃止 → フラットなカードグリッドに |
| 工程別見積精度（カード型） | 統合 → バイアス分析チャートに |
| 異常値検出セクション | 統合 → アラートバーに |
| 要注意タスク一覧 | 統合 → アラートバーの展開部に |
| メンバー比較棒グラフ (300px) | 置換 → メンバー負荷分布（コンパクト） |
| 個別メンバードーナツチャート | 廃止 → ステータス分布ドーナツ1つに統合 |
| キャパシティ分析セクション | 統合 → アラートバー + KPIバッジに |
| インサイト（AIライク）セクション | 統合 → 各チャートカードの下部インサイトに分散 |

## 技術方針

- **Canvas 2D API** を継続使用（外部チャートライブラリなし）
- チャートカードを**統一コンポーネント関数**で生成
- デザインシステムの CSS 変数を全面適用（Bootstrap系カラーを排除）
- レスポンシブ: デスクトップ 2列、タブレット/モバイル 1列
- リサイズ時のCanvas再描画対応

## モックアップファイル

- `index.html` — インタラクティブなHTMLモックアップ（ブラウザで直接確認可能）
  - サンプルデータで全8チャートを描画
  - アラートバーの展開/閉じ動作
  - 詳細セクションの折りたたみ動作
  - ウィンドウリサイズでチャート再描画
