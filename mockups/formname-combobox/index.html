帳票名 Combobox（モック）

入力で絞り込み、選択も新規入力も同じフィールドで完結。iPhone / PC 両対応。

📝 見積登録
現在の値: （未入力）

操作

  • タップ / フォーカス 候補一覧を表示
  • 文字入力 部分一致でインクリメント絞り込み（一致箇所をハイライト）
  • ▼ ボタン 全候補を再表示
  • ↑ ↓ 移動 / Enter 確定 / Esc 閉じる（PC）
  • 候補にない文字列を入力した場合は「＋ 新規として登録」が末尾に出現 → タップで確定

iOS / モバイル対策

  • input の font-size: 16px 以上で自動ズーム回避
  • 候補は pointerdown で確定（blur より先に発火させて「タップが効かない」問題を回避）
  • 仮想キーボードが候補を覆う場合、自動で input の上側に展開
  • 候補リストは max-height: 280px + スクロール（iOS datalist で候補が少ししか出ない問題を回避）
