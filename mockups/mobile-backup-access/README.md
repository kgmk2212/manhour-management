# モバイル版バックアップ復元アクセス改善

## 背景

モバイル表示時、バックアップファイルからの復元にサイドバーを開く必要があり操作性が悪い。
サイドバーを開かずにバックアップ・復元・変更履歴にアクセスする方法を検討。
